Dicas de uso do VS Code

Para selecionar várias tags

  1. Selecione apenas a TAG de abertura
  2. pressione Ctlr+D para selecionar a mesma TAG da próxima linha. Ao repetir esta ação, é possível selecionar UMA A UMA ,todas a TAGs do mesmo tipo no código inteiro, para realizar quaisquer ajustes. Pois isso possibilita navegar com as setas do teclado pra frente ou para trás. Caso queria selecionar TODAS as TAGs de uma só vez, acione "Ctrl+F2".
  3. Pode ser acrescentado um "href" ou removido um trecho errado dessa lista de tags selecionadas.
  4. Para criar massa de dados tipo texto, digite "lorem" e deixe o autocomplete do VSCode preencher com um parágrafo.
