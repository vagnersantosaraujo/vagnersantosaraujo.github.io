Organização dos diretórios

É importante padronizar os diretórios do projetos, abaixo, algumas sugestões para organizar os diretórios:

  1. O diretório "assets" é o diretório onde ficam os "ativos" do projeto como: fontes, ícones, imagens e vídeos.
  2. O diretório "src" é o diretório onde ficam os arquivos que são utilizados para criar o projeto.
  3. O diretório "dist" é o diretório onde ficam os arquivos que são utilizados para o projeto.
  4. O diretório "docs" é o diretório onde ficam os documentos do projeto.
  5. O diretório "tests" é o diretório onde ficam os testes do projeto.
  6. O diretório "node_modules" é o diretório onde ficam os módulos do projeto.
  7. O diretório "public" é o diretório onde ficam os arquivos que são utilizados para o projeto.
  8. O diretório "src/components" é o diretório onde ficam os arquivos que são utilizados para o projeto.
  9. O diretório "src/controllers" é o diretório onde ficam os arquivos que são utilizados para o projeto.
  10. O diretório "src/models" é o diretório onde ficam os arquivos que são utilizados para o projeto.
  11. O diretório "src/services" é o diretório onde ficam os arquivos que são utilizados para o projeto.
  12. O diretório "src/views" é o diretório onde ficam os arquivos que são utilizados para o projeto.
  13. O diretório "src/views/components" é o diretório onde ficam os arquivos que são utilizados para o projeto.
  14. O diretório "src/views/controllers" é o diretório onde ficam os arquivos que são utilizados para o projeto.
  15. O diretório "src/views/models" é o diretório onde ficam os arquivos que são utilizados para o projeto.
  16. O diretório "src/views/services" é o diretório onde ficam os arquivos que são utilizados para o projeto.
  17. O diretório "src/views/views" é o diretório onde ficam os arquivos que são utilizados para o projeto.
  18. O diretório "src/views/views/components" é o diretório onde ficam os arquivos que são utilizados para o projeto.
