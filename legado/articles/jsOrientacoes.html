No JS ou JavaScript, se houver chaves, trata-se de um objeto.

Existem 3 formas de criar variárveis no JavaScript, são elas:

  • var - possui problemas de escopo, como vazamento de for, if e while. Isso ocorre por que o VAR não precisa definir a varirável
  • let - o LET foi criado para resolver o problema de escopo do VAR.
  • const - a mais interessante, pois trava possíveis alterações nas constantes. SEMPRE USE CONST!

Para contruir uma variável basta usar a sintaxe title = "Harry Potter"; ou ainda const url = "https://caminho.com/imagem";

Para construir um objeto, basta usar a sintaxe 

        const movie = {
            const title = "Herry Potter";
            const url = "https://www.caminho.com/imagem";
        };

Toda característica de um objeto é um atributo ou propriedade.

Para acessar uma propriedade de um objeto, basta usar o ponto, exemplo: movie.titulo

Ações são métodos de um objeto, abaixo um exemplo 

const listBanner = Document.querySelector(".list-banners");

Onde:
const é a variável
(lê-se "A variável do tipo const") listBanner é o obejeto
(lê-se "A variável ListBanner do tipo const") = é o operador de atribuição (lê-se "A variável ListBanner do tipo const recebe")
DocumentÉ um objeto, ele representa o HTML dentro do JavaScript. (lê-se "A variável ListBanner do tipo const recebe do objeto document") .querySelector realiza uma ação, ou seja, busca um elemento na página. Também pode ser descrito como a "execução de um método do objeto (lê-se "A variável ListBanner do tipo const, recebe do objeto document a busca do seletor")
(".list-banners")tudo que estiver entre os parenteses será usado para buscar no document (no HTML). Lê-se "A variável ListBanner do tipo const, recebe do objeto document a busca do seletor list-banners que está na página HTML")

Use o . (ponto) para acessar algo que está dento do Document (dentro do HTML).Veja o exemplo: document.createElemnt(). Note que estamos solicitando ao HTML para criar algo, e que neste momento, é algo vazio, pois não dissemos o que deve ser criado, para isso, é necessário preenhcer os parenteses

Os parenteses são obrigatórios para a execução, para a ação ou sisplesmente método! Exemplo: se os parenteses forem preenchidos com "h1", assim que for executado, será criado um título h1 no HTML

As variáveis e funções (sempre use o tipo const) podem armazenar o resultado da execução de criação de objetos. Desde que ela seja declarada antes do objeto document. Exemplo:
const img = document.createElement("img");
Onde a variável const está armazenando o criado pelo método createElement.
