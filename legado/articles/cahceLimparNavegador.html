Limpar cache do navegador

Três atalhos para limpara o cache do navagador, pois o navegador guarda uma versão antiga do site, veja:

  1. CTRL + R (limpa o cache)
  2. CTRL + SHIFT + R (limpa o chache de forma profunda)
  3. CTRL + F5 (também limpa mas, é necessário pesquisar mais)

Usar o F5 para atualizar a página pode funcionar porém, o navegador pode estar exibindo uma versão antiga do site.
