Orientações de bom uso do HTML e CSS

  1. Não use TAGs vazias. Isso é gato!
  2. Não use TAGs desnecessárias.
  3. Antes de começar um novo arquivo CSS, use um reset CSS, um exemplo bem famoso é o Reset CSS do Erick Meyer
  4. A propriedade "display: inline" (que é padrão, mesmo que não esteja escrita no CSS), não permite usar duas propriedade, isto é, a largura e altura não funcionam juntas, então, é necessário usar "display: inline-block" que que as definições de altura e largura tenham efeito no navegador. O display inline também possui o comportamento de texto
  5. Alguma tags do HTML ocoupam a linha inteira, uma delas é o elemento NAV, que é diferente d de um "a" que só ocupa o esço do seu conteúdo.
  6. Para facilitar a leitura do código CSS, é interessante criar um diretório componets para criar vários aquivos CSS, um para cada tipo TAG ou ID ou Classe, o que é ótimo para a manutenção. Porém, isso pode deixar mais lento!
  7. Commits com comentários só em inglês!
  8. Se precisar formar textos, é necessário usar formatar os elemento H1, H2 e etc.
  9. ID não se repete! Use class se precisa aplicar o estilo em vários lugares
